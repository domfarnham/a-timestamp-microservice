A Timestamp Microservice By dom.farnham()

A microservice which converts natural date strings to Unix timestamps and vice-versa.

Example usage:

https://a-timestamp-microservice.glitch.me/December%2015,%202015

https://a-timestamp-microservice.glitch.me/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
